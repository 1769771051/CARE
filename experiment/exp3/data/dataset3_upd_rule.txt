# 第三章 车辆及路面电车的通行方式
## 第一节 通则
（通则）
第十六条 道路上车辆及路面电车的通行方式，依照本章规定执行。
2. 就本章规定的适用而言，以汽车或机动自行车牵引其他车辆时，被牵引的车辆视为牵引该车辆的汽车或机动自行车的一部分。
3. 本章中与交叉路口交通相关的规定，不适用于在干线车道通行的汽车。
4. 就本章规定的适用而言，设有自行车道的道路上，自行车道与自行车道以外的车道部分，均分别视为一条车道。

（通行区分）
第十七条 车辆在设有人行道或路肩（以下本条及次条第一项中称为“人行道等”）与车道区分的道路上，必须在车道通行。但为进出道路外的设施或场所而不得已横穿人行道等时，或依据第四十七条第三项或第四十八条规定，为在人行道等临时停车或停车而必要范围内在人行道等通行时，不在此限。
2. 在前项但书规定的情形下，车辆必须在进入人行道等直前临时停车，且不得妨碍行人通行。
3. 除特定小型机动自行车（指机动自行车中符合第二条第一项第十号乙的车辆，以下同）、二轮或三轮自行车以及其他车体大小及结构符合内阁府令规定标准、不会妨碍自行车道上其他车辆通行的车辆（不含带边车的此类车辆及牵引其他车辆的车辆）外，其他车辆不得在自行车道通行。但为进出道路外的设施或场所而不得已时，可横穿自行车道。
4. 车辆必须在道路（设有人行道等与车道区分的道路时指车道，以下至第九节之二同）的中央（轨道靠近道路侧端设置时，指该道路除去轨道敷设部分的中央；设有道路标识等规定的中心线时，指设置该中心线的道路部分的中央，以下同）左侧的部分（以下称为“左侧部分”）通行。
5. 车辆在下列情形下，不受前项规定限制，可全部或部分超出道路中央右侧的部分（以下称为“右侧部分”）通行。此时，除第一号情形外，车辆应尽量减少超出的范围。
一、该道路为单向通行（指道路上车辆通行禁止以特定方向进行，以下同）时；
二、该道路左侧部分的宽度不足以供该车辆通行情时；
三、该车辆因道路损坏、道路施工或其他障碍无法在道路左侧部分通行时；
四、在左侧部分宽度不足六米的道路上，试图超越其他车辆时（仅限能够看清道路右侧部分且无妨碍对向交通之虞，且未被道路标识等禁止为超越而超出右侧部分通行的情形）；
五、在陡坡的弯道附近，道路标识等指定了通行方式，且车辆按照该指定通行时。
6. 车辆不得进入安全地带或道路标识等标明不得供车辆通行的其他道路部分。

（罚则 第一至三项及第六项：第一百十九条第一项第六号 第四项：第一百十七条之二第一项第四号、第一百十七条之二之二第一项第八号甲、第一百十九条第一项第六号）

（特例特定小型机动自行车的人行道通行）
第十七条之二 特定小型机动自行车中，符合下列各项规定且未牵引其他车辆的（不含可远程操控通行的，以下本条及次条中称为“特例特定小型机动自行车”），不受前条第一项规定限制，若道路标识等允许特例特定小型机动自行车在人行道通行，可在该人行道通行。但警察官等为确保行人安全，认为有必要指示不得在该人行道通行时，不在此限。
一、在人行道等通行期间，通过内阁府令规定的方式标明该特定小型机动自行车可在人行道等通行；
二、在符合前号规定标明的情形下，因车体结构限制，不得超过内阁府令规定的、不会妨碍人行道等上行人通行的速度；
三、除前两项规定外，车体结构符合内阁府令规定的、不会妨碍人行道等上行人通行的标准。
2. 在前项情形下，特例特定小型机动自行车必须在该人行道中央靠近车道的部分（设有普通自行车通行指定部分时，指该普通自行车通行指定部分）低速行驶；若其行驶将妨碍行人通行，必须临时停车。但对于普通自行车通行指定部分，当无行人正在或试图在该部分通行时，可根据人行道状况以安全的速度和方式行驶。

（罚则 第二项：第一百二十一条第一项第八号）

（特例特定小型机动自行车等的路肩通行）
第十七条之三 特例特定小型机动自行车及轻型车辆，不受第十七条第一项规定限制，除显著妨碍行人通行的情形外，可在道路左侧部分设置的路肩（不含以道路标识划分禁止特例特定小型机动自行车及轻型车辆通行的部分）通行。
2. 在前项情形下，特例特定小型机动自行车及轻型车辆必须以不妨碍行人通行的速度和方式行驶。

（罚则 第二项：第一百二十一条第一项第八号）

（靠左通行等）
第十八条 车辆（不含无轨电车）除在设有车辆通行带的道路通行外，汽车及一般机动自行车（指机动自行车中符合第二条第一项第十号甲的车辆，以下同）应靠近道路左侧通行，特定小型机动自行车及轻型车辆（以下称为“特定小型机动自行车等”）应靠近道路左侧端通行。但超车时、依据第二十五条第二项或第三十四条第二项或第四项规定靠近道路中央或右侧端时，或因道路状况及其他不得已的情形时，不在此限。
2. 车辆在前项规定的无人行道与车道区分的道路通行等情形下，从行人侧方通过时，必须与行人保持安全间距或低速行驶。
3. 车辆（不含特定小型机动自行车等）在超越同一方向行驶的特定小型机动自行车等（不含在人行道或自行车道通行的）右侧时（不含超车情形），若与该特定小型机动自行车等之间无足够间距，必须根据间距保持安全速度行驶。
4. 在前项规定的情形下，特定小型机动自行车等应尽量靠近道路左侧端通行。

（罚则 第二项：第一百十九条第一项第六号 第三项：第一百十七条之二第一项第四号、第一百十七条之二之二第一项第八号乙、第一百十九条第一项第六号 第四项：第一百二十条第一项第二号）

（禁止轻型车辆并排行驶）
第十九条 轻型车辆不得与其他轻型车辆并排行驶。

（罚则 第一百二十一条第一项第八号）

（车辆通行带）
第二十条 车辆在设有车辆通行带的道路上，必须在从道路左侧端数起的第一条车辆通行带通行。但汽车（不含小型特殊汽车及道路标识等指定的汽车）在该道路左侧部分（单向通行道路时指该道路）设有三条以上车辆通行带时，可依据政令规定，根据车速在最右侧车辆通行带以外的车辆通行带通行。
2. 车辆在设有车辆通行带的道路上，若道路标识等指定了与前项规定不同的通行区分，必须按照该通行区分在相应车辆通行带通行。
3. 车辆在超车时、依据第十八条第四项、第二十五条第一项或第二项、第三十四条第一至五项或第三十五条之二规定靠近道路左侧端、中央或右侧端时、按照第三十五条第一项规定通行时、依据第二十六条之二第三项规定在当前车辆通行带继续通行时、依据第四十条第二项规定临时让行时，或因道路状况及其他不得已的情形时，可不遵守前两项规定。此时，超车时必须在当前车辆通行带紧邻的右侧车辆通行带通行。

（罚则 第一百二十条第一项第三号、同条第三项）

（公交线路车等优先通行带）
第二十条之二 道路上设有经道路标识等标明为道路运输法第九条第一项规定的一般旅客汽车运输业者用于同法第五条第一项第三号规定的定期线路运行的汽车及其他政令规定的汽车（以下本条中称为“公交线路车等”）优先通行带的，汽车（不含公交线路车等，以下本条同）若因道路交通拥堵，在公交线路车等从后方接近时无法驶离该通行带，则不得在该通行带通行；若已在该通行带通行，当公交线路车等从后方接近时，必须立即驶离该通行带，不得妨碍其正常运行。但依据本法其他规定必须在该通行带通行，或因道路状况及其他不得已的情形时，不在此限。
2. 前条第一项本文的规定，不适用于在上述优先通行带紧邻的右侧车辆通行带或道路部分通行的汽车。

（罚则 第一项：第一百二十条第一项第三号、同条第三项）

（轨道敷设内的通行）
第二十一条 车辆（不含无轨电车，以下本条及次条第一项同）除为左转、右转、横穿或掉头而穿越轨道敷设，或为防止危险不得已的情形外，不得在轨道敷设内通行。
2. 车辆在下列情形下，不受前项规定限制，可在轨道敷设内通行。此时，车辆不得妨碍路面电车通行。
一、该道路左侧部分除去轨道敷设后的宽度不足以供该车辆通行情时；
二、该车辆因道路损坏、道路施工或其他障碍无法在道路左侧部分除去轨道敷设后的部分通行时；
三、经道路标识等允许通行的汽车通行时。
3. 在轨道敷设内通行的车辆，当路面电车从后方接近时，必须立即驶离轨道敷设，或与该路面电车保持必要距离，不得妨碍其正常运行。

（罚则 第一百二十一条第一项第八号）

## 第二节 速度
（最高速度）
第二十二条 车辆在道路标识等指定最高速度的道路上，不得超过该最高速度行驶；在其他道路上，不得超过政令规定的最高速度行驶。
2. 路面电车或无轨电车，在不超过依据轨道法（大正十年法律第七十六号）第十四条（含同法第三十一条准用的情形，第六十二条同）规定的命令所定最高速度的范围内，在道路标识等指定最高速度的道路上，不得超过该最高速度行驶；在其他道路上，不得超过该命令规定的最高速度行驶。

（罚则 第一百十八条第一项第一号、同条第三项）

（对最高速度违法行为相关车辆使用者的指示）
第二十二条之二 车辆驾驶人在执行该车辆使用者（不含该车辆驾驶人本人，以下本条同）的业务过程中，实施了违反前条规定的行为（以下本条及第七十五条之二第一项中称为“最高速度违法行为”），且该车辆使用者未对该车辆采取必要的运营管理以防止最高速度违法行为时，管辖该车辆使用所在地的公安委员会可指示该车辆使用者对驾驶人进行指导或建议，防止发生构成最高速度违法行为的驾驶行为，并采取其他必要措施防止最高速度违法行为。
2. 若依据前项规定接受指示的车辆使用者是依据道路运输法规定的汽车运输业者、依据货物利用运输业法（平成元年法律第八十二号）规定经营第二种货物利用运输业的经营者，或依据轨道法规定的轨道经营者（仅限运营无轨电车的），公安委员会必须在事先与负责监督该业务的行政机关协商后，作出该指示。

（最低速度）
第二十三条 汽车在道路标识等指定最低速度的道路（不含第七十五条之四规定的高速公路干线车道）上，除依据法令规定减速或为防止危险不得已的情形外，不得低于该最低速度行驶。

（禁止急刹车）
第二十四条 车辆等的驾驶人，除为防止危险不得已的情形外，不得突然停止车辆等或急剧减速而采取急刹车。

（罚则 第一百十七条之二第一项第四号、第一百十七条之二之二第一项第八号丙、第一百十九条第一项第三号）

## 第三节 横穿等
（驶出道路时的方式）
第二十五条 车辆为驶出道路而左转时，必须提前尽量靠近道路左侧端，并低速行驶。
2. 车辆（不含特定小型机动自行车等及无轨电车）为驶出道路而右转时，必须提前尽量靠近道路中央（单向通行道路时指道路右侧端），并低速行驶。
3. 试图为驶出道路而左转或右转的车辆，依据前两项规定，试图靠近道路左侧端、中央或右侧端并以手势或方向指示器示意时，后方车辆除必须突然改变速度或方向的情形外，不得妨碍该示意车辆变更车道。

（罚则 第一及二项：第一百二十一条第一项第八号 第三项：第一百二十条第一项第二号）

（禁止横穿等）
第二十五条之二 车辆若有妨碍行人或其他车辆等正常交通之虞，不得为进出道路外的设施或场所而左转、右转、横穿、掉头或倒车。
2. 车辆在道路标识等禁止横穿、掉头或倒车的道路部分，不得实施该禁止行为。

（罚则 第一项：第一百十九条第一项第六号 第二项：第一百二十条第一项第四号、同条第三项）

## 第四节 超车等
（保持车距）
第二十六条 车辆等在同一车道紧随其他车辆等行驶时，必须保持必要的安全距离，以避免前车突然停止时发生追尾碰撞。

（罚则 第一百十七条之二第一项第四号、第一百十七条之二之二第一项第八号丁、第一百十九条第一项第四号、第一百二十条第一项第二号）

（禁止随意变更车道）
第二十六条之二 车辆不得随意变更车道。
2. 车辆若变更车道可能导致后方同向行驶的车辆等突然改变速度或方向，不得变更车道。
3. 车辆在车辆通行带内行驶时，若该车辆通行带由标明禁止变更车道的道路标识划分，除下列情形外，不得跨越该道路标识变更车道。
一、依据第四十条规定靠近道路左侧或右侧，或因道路损坏、道路施工或其他障碍无法在当前车辆通行带通行时；
二、为遵守第四十条规定，或因道路损坏、道路施工或其他障碍无法通行的车辆通行带，试图按照通行区分规定通行时。

（罚则 第二项：第一百十七条之二第一项第四号、第一百十七条之二之二第一项第八号戊、第一百二十条第一项第二号 第三项：第一百二十条第一项第三号、同条第三项）

（被其他车辆追上时的义务）
第二十七条 车辆（不含道路运输法第九条第一项规定的一般旅客汽车运输业者用于同法第五条第一项第三号规定的定期线路运行或同法第三条第二号规定的特定旅客汽车运输业务的汽车（以下称为“客运汽车”）及无轨电车）被最高速度（指依据第二十二条第一项规定的政令所定最高速度，以下本条同）更高的车辆追上时，在该追上的车辆完成超车前，不得加速。被最高速度相同或更低的车辆追上，且试图继续以低于该追上车辆的速度行驶时，亦同。
2. 车辆（不含客运汽车及无轨电车）除在设有车辆通行带的道路通行外，被最高速度更高的车辆追上，且道路中央（单向通行道路时指道路右侧端，以下本项同）与自身之间无足够空间供该追上车辆通行时，不受第十八条第一项规定限制，必须尽量靠近道路左侧端让行。被最高速度相同或更低的车辆追上，且道路中央与自身之间无足够空间供该追上车辆通行，且试图继续以低于该追上车辆的速度行驶时，亦同。

（罚则 第一百二十条第一项第二号）

（超车方式）
第二十八条 车辆试图超越其他车辆时，必须从被超越车辆（以下本节中称为“前车”）的右侧通行。
2. 车辆试图超越其他车辆时，若前车依据第二十五条第二项或第三十四条第二项或第四项规定靠近道路中央或右侧端通行，不受前项规定限制，必须从其左侧通行。
3. 车辆试图超越路面电车时，必须从追上的路面电车左侧通行。但轨道靠近道路左侧端设置时，不在此限。
4. 在前三项情形下，试图超车的车辆（次条中称为“后车”）必须充分注意对向或后方交通以及前车或路面电车前方的交通，并根据前车或路面电车的速度、车道及道路状况，以尽可能安全的速度和方式行驶。

（罚则 第一及四项：第一百十七条之二第一项第四号、第一百十七条之二之二第一项第八号己、第一百十九条第一项第六号 第二及三项：第一百十九条第一项第六号）

（禁止超车的情形）
第二十九条 后车在前车试图超越其他汽车或无轨电车时，不得开始超车。

（罚则 第一百十九条第一项第六号）

（禁止超车的地点）
第三十条 车辆在道路标识等禁止超车的道路部分及下列其他道路部分，不得为超越其他车辆（不含特定小型机动自行车等）而变更车道或从前车侧方通过。
一、道路弯道附近、上坡顶端附近或陡坡下坡路段；
二、隧道（仅限未设有车辆通行带的道路部分）；
三、交叉路口（不含该车辆在第三十六条第二项规定的优先道路通行时，该优先道路上的交叉路口）、铁路道口、人行横道或自行车横穿带，以及从这些设施的侧端向前三十米以内的部分。

（罚则 第一百十九条第一项第五号、同条第三项）

（遇停站路面电车时的停止或低速行驶）
第三十一条 车辆追上因乘客上下车而停驶的路面电车时，必须在该路面电车后方停车，直至乘客上下车完毕，且无从该路面电车下车后在车辆前方横穿或试图横穿路面电车左侧的人员为止。但设有为保障上下路面电车人员安全的安全地带时，或无上下路面电车的人员且能与该路面电车左侧保持一点五米以上间距时，可低速从该路面电车左侧通过。

（罚则 第一百十九条第一项第六号）

（保护客运汽车发车）
第三十一条之二 客运汽车在车站因乘客上下车停驶后，为发车而试图变更车道并以手势或方向指示器示意时，后方车辆除必须突然改变速度或方向的情形外，不得妨碍该示意客运汽车变更车道。

（罚则 第一百二十条第一项第二号）

（禁止强行插入等）
第三十二条 车辆追上依据法令规定、警察官命令，或为防止危险而停止、即将停止并低速行驶的车辆等，或紧随其后停止、低速行驶的车辆等时，不得从其侧方通过并强行插入前方，或横穿其前方。

（罚则 第一百二十条第一项第二号）

## 第五节 铁路道口的通过
（铁路道口的通过）
第三十三条 车辆等试图通过铁路道口时，必须在铁路道口直前（设有道路标识等规定的停止线时，指该停止线直前，以下本项同）停车，确认安全后，方可通行。但按照信号灯指示行驶时，可不需在铁路道口直前停车。
2. 车辆等试图通过铁路道口时，若道口栏杆即将关闭、正在关闭，或道口警报器正在报警，不得进入该铁路道口。
3. 车辆等的驾驶人若因故障或其他原因无法在铁路道口驾驶车辆等时，必须立即发出紧急信号等，采取措施将铁路道口有车辆等因故障或其他原因停驶的情况通知铁路或轨道工作人员或警察官，并采取必要措施将该车辆等移至铁路道口以外的地点。

（罚则 第一及二项：第一百十九条第一项第五号、同条第三项）

## 第六节 交叉路口的通行方式等
（左转或右转）
第三十四条 车辆左转时，必须提前尽量靠近道路左侧端，并尽量沿道路左侧端（设有道路标识等指定通行部分时，指该指定部分）低速行驶。
2. 汽车、一般机动自行车或无轨电车右转时，必须提前尽量靠近道路中央，并沿交叉路口中心附近的内侧（设有道路标识等指定通行部分时，指该指定部分）低速行驶。
3. 特定小型机动自行车等右转时，必须提前尽量靠近道路左侧端，并沿交叉路口侧端低速行驶。
4. 汽车、一般机动自行车或无轨电车在单向通行道路上右转时，不受第二项规定限制，必须提前尽量靠近道路右侧端，并沿交叉路口中心内侧（设有道路标识等指定通行部分时，指该指定部分）低速行驶。
5. 一般机动自行车不受第二项及前项规定限制，在道路标识等进行交通疏导的交叉路口，若指定一般机动自行车右转应沿交叉路口侧端通行的道路，以及道路左侧部分（单向通行道路时指道路）设有三条以上车辆通行带的其他道路（以下本项中称为“多车道道路”）上右转时（仅限在进行交通疏导的交叉路口右转），必须提前尽量靠近道路左侧端，并沿交叉路口侧端低速行驶。但多车道道路上，若道路标识等提前指定一般机动自行车在进行交通疏导的交叉路口右转应靠近道路中央或右侧端时，不在此限。
6. 试图左转或右转的车辆，依据前各项规定，试图靠近道路左侧端、中央或右侧端并以手势或方向指示器示意时，后方车辆除必须突然改变速度或方向的情形外，不得妨碍该示意车辆变更车道。

（罚则 第一至五项：第一百二十一条第一项第八号 第六项：第一百二十条第一项第二号）

（指定通行区分）
第三十五条 车辆（不含特定小型机动自行车等及在交叉路口左转或右转时，一般机动自行车应遵守前条第五项本文规定的一般机动自行车）在设有车辆通行带的道路上，若道路标识等指定了交叉路口行驶方向的通行区分，不受前条第一、二及四项规定限制，必须按照该通行区分在相应车辆通行带通行。但为遵守第四十条规定，或因道路损坏、道路施工或其他不得已的情形时，不在此限。
2. 前条第六项的规定，准用于车辆为按照前项通行区分通行而试图变更车道并以手势或方向指示器示意的情形。

（罚则 第一项：第一百二十条第一项第三号、同条第三项 第二项：第一百二十条第一项第二号）

（环形交叉路口的左转等）
第三十五条之二 车辆在环形交叉路口左转或右转时，不受第三十四条第一至五项规定限制，必须提前尽量靠近道路左侧端，并尽量沿环形交叉路口侧端（设有道路标识等指定通行部分时，指该指定部分）低速行驶。
2. 车辆在环形交叉路口直行或掉头时，必须提前尽量靠近道路左侧端，并尽量沿环形交叉路口侧端（设有道路标识等指定通行部分时，指该指定部分）低速行驶。

（罚则 第一百二十一条第一项第八号）

（交叉路口与其他车辆等的关系等）
第三十六条 车辆等在未进行交通疏导的交叉路口，除适用次项规定的情形外，必须按照下列区分，不得妨碍各号所列车辆等的通行。
一、属于车辆的，不得妨碍从左侧交叉道路（以下称为“交叉道路”）行驶而来的车辆及在交叉道路通行的路面电车；
二、属于路面电车的，不得妨碍从左侧交叉道路行驶而来的路面电车。
2. 车辆等在未进行交通疏导的交叉路口，除其行驶的道路为优先道路（指经道路标识等指定为优先道路的道路，及在该交叉路口设有规范该道路车辆通行的道路标识等规定的中心线或车辆通行带的道路，以下同）的情形外，若交叉道路为优先道路，或交叉道路的宽度明显宽于其行驶的道路宽度，不得妨碍在该交叉道路通行的车辆等的通行。
3. 车辆等（不含在优先道路通行的车辆等）试图进入未进行交通疏导的交叉路口时，若交叉道路为优先道路，或交叉道路的宽度明显宽于其行驶的道路宽度，必须低速行驶。
4. 车辆等试图进入交叉路口及在交叉路口内通行时，必须根据交叉路口的状况，特别注意在交叉道路通行的车辆等、从对向行驶而来并右转的车辆等，以及在该交叉路口或其附近横穿道路的行人，并以尽可能安全的速度和方式行驶。

（罚则 第一项：第一百二十条第一项第二号 第二至四项：第一百十九条第一项第六号）

第三十七条 车辆等在交叉路口右转时，若有车辆等试图在该交叉路口直行或左转，不得妨碍其通行。

（罚则 第一百二十条第一项第二号）

（环形交叉路口与其他车辆等的关系等）
第三十七条之二 车辆等在环形交叉路口，不受第三十六条第一、二项及前条规定限制，不得妨碍在该环形交叉路口内通行的车辆等的通行。
2. 车辆等试图进入环形交叉路口时，不受第三十六条第三项规定限制，必须低速行驶。
3. 车辆等试图进入环形交叉路口及在环形交叉路口内通行时，不受第三十六条第四项规定限制，必须根据环形交叉路口的状况，特别注意试图进入该环形交叉路口的车辆等、在该环形交叉路口内通行的车辆等，以及在该环形交叉路口或其附近横穿道路的行人，并以尽可能安全的速度和方式行驶。

（罚则 第一百十九条第一项第六号）

## 第六节之二 为保护横穿行人等的通行方式
（人行横道等上行人等的优先通行权）
第三十八条 车辆等接近人行横道或自行车横穿带（以下本条中称为“人行横道等”）时，除明确无行人或自行车（以下本条中称为“行人等”）试图通过该人行横道等横穿前方车道的情形外，必须以能够在该人行横道等直前（设有道路标识等规定的停止线时，指该停止线直前，以下本项同）停车的速度行驶。此时，若有行人等正在或试图通过该人行横道等横穿前方车道，必须在该人行横道等直前临时停车，且不得妨碍其通行。
2. 车辆等若遇到在人行横道等（不含车辆等通过时，信号灯显示或警察官等人手势等禁止行人等通过该人行横道等横穿的情形，次项同）或其直前停驶的车辆等，试图从该停驶车辆等侧方通过并驶至其前方时，必须在驶至前方前临时停车。
3. 车辆等在人行横道等及其侧端向前三十米以内的道路部分，除符合第三十条第三号规定的情形外，不得从前方行驶的其他车辆等（不含特定小型机动自行车等）侧方通过并驶至其前方。

（罚则 第一百十九条第一项第五号、同条第三项）

（无人行横道交叉路口的行人优先通行权）
第三十八条之二 车辆等在交叉路口或其附近无人行横道的地点，若有行人横穿道路，不得妨碍其通行。

（罚则 第一百十九条第一项第六号）

## 第七节 紧急车辆等
（紧急车辆的通行区分等）
第三十九条 紧急车辆（指消防车、救护车及其他政令规定的车辆，为执行紧急任务，依据政令规定正在行驶的，以下同）除第十七条第五项规定的情形外，因超车或其他不得已的必要时，不受同条第四项规定限制，可全部或部分超出道路右侧部分通行。
2. 紧急车辆即使依据法令规定应停车，也无需停车。此时，必须注意其他交通并低速行驶。

（紧急车辆的优先通行权）
第四十条 交叉路口或其附近有紧急车辆接近时，路面电车应避开交叉路口，车辆（不含紧急车辆，以下本条同）应避开交叉路口，并靠近道路左侧（单向通行道路上，靠近左侧会妨碍紧急车辆通行时，指道路右侧，次项同）临时停车。
2. 在前项规定以外的地点有紧急车辆接近时，车辆应靠近道路左侧让行。

（罚则 第一百二十条第一项第二号）

（紧急车辆等的特例）
第四十一条 紧急车辆不适用第八条第一项、第十七条第六项、第十八条第一至三项、第二十条第一及二项、第二十条之二、第二十五条第一及二项、第二十五条之二第二项、第二十六条之二第三项、第二十九条、第三十条、第三十四条第一、二及四项、第三十五条第一项以及第三十八条第一项前段及第三项的规定。
2. 除前项规定外，紧急车辆在查处违反第二十二条规定的车辆等时，不适用该条规定。
3. 专门从事交通管理的汽车，符合内阁府令规定的，不适用第十八条第一项、第二十条第一及二项、第二十条之二以及第二十五条之二第二项的规定。
4. 道路维护作业用汽车（指专门用于道路维护、修缮等的汽车，符合政令规定的，以下第七十五条之九同）在依据政令规定从事道路维护、修缮等作业时，不适用第十七条第四及六项、第十八条第一项、第二十条第一及二项、第二十条之二、第二十三条以及第二十五条之二第二项的规定。

（消防车的优先通行权等）
第四十一条之二 交叉路口或其附近有消防车（指除消防车以外用于消防业务的车辆，为执行消防任务，依据政令规定正在行驶的，以下本条及第七十五条之二十二第二项同）接近时，车辆等（车辆不含紧急车辆及消防车）应避开交叉路口临时停车。
2. 在前项规定以外的地点有消防车接近时，车辆（不含紧急车辆及消防车）不得妨碍该消防车通行。
3. 第三十九条的规定准用于消防车。
4. 消防车不适用第八条第一项、第十七条第六项、第十八条、第二十条第一及二项、第二十五条第一及二项、第二十五条之二第二项、第二十六条之二第三项、第二十九条、第三十条、第三十四条第一至五项、第三十五条第一项、第三十八条第一项前段及第三项、第四十条第一项、第六十三条之六以及第六十三条之七的规定。

（罚则 第一及二项：第一百二十条第一项第二号）

## 第八节 低速行驶及临时停车
（应低速行驶的地点）
第四十二条 车辆等在道路标识等指定应低速行驶的道路部分通行时，及在下列其他情形下，必须低速行驶。
一、试图进入左右视线受阻的交叉路口，或在交叉路口内左右视线受阻的部分通行时（不含该交叉路口正在进行交通疏导及在优先道路通行的情形）；
二、在道路弯道附近、上坡顶端附近或陡坡下坡路段通行时。

（罚则 第一百十九条第一项第五号、同条第三项）

（指定地点的临时停车）
第四十三条 车辆等在未进行交通疏导的交叉路口或其直前，若道路标识等指定应临时停车，必须在道路标识等规定的停止线直前（未设有道路标识等规定的停止线时，指交叉路口直前）临时停车。此时，该车辆等除符合第三十六条第二项规定的情形外，不得妨碍在交叉道路通行的车辆等的通行。

（罚则 第一百十九条第一项第五号、同条第三项）

## 第九节 临时停车及停车
（禁止临时停车及停车的地点）
第四十四条 车辆在道路标识等禁止临时停车及停车的道路部分及下列其他道路部分，除依据法令规定、警察官命令，或为防止危险临时停车的情形外，不得临时停车或停车。
一、交叉路口、人行横道、自行车横穿带、铁路道口、轨道敷设内、坡顶附近、陡坡或隧道；
二、交叉路口侧端或道路弯道起五米以内的部分；
三、人行横道或自行车横穿带前后侧端起各五米以内的部分；
四、设有安全地带的道路上，该安全地带左侧的部分及该部分前后侧端起各十米以内的部分；
五、设有客运汽车车站或无轨电车、路面电车站标识柱或标识牌的位置起十米以内的部分（仅限属于该车站运营线路的客运汽车、无轨电车或路面电车的运营时间内）；
六、铁路道口前后侧端起各十米以内的部分。
2. 前项规定不适用于下列情形。
一、客运汽车或无轨电车在其所属运营线路的车站，为乘客上下车临时停车，或为调整运营时间停车时；
二、用于旅客运输的汽车（不含客运汽车，第四十九条之三第一项同）在客运汽车车站或无轨电车、路面电车站，为乘客上下车临时停车，或为调整运营时间停车时（仅限该临时停车或停车对保障当地居民生活所需的旅客运输有益，且根据道路或交通状况无妨碍，依据内阁府令规定，经道路运输法第九条第一项规定的一般旅客汽车运输业者、公安委员会及其他内阁府令规定的与该临时停车或停车相关的人员协商一致，并由公安委员会公示的情形）。

（罚则 第一项：第一百十九条之二之四第一项第一号、同条第三项、第一百十九条之三第一项第一号、同条第三项）

（禁止停车的地点）
第四十五条 车辆在道路标识等禁止停车的道路部分及下列其他道路部分，不得停车。但依据公安委员会规定，经警察署长许可的，不在此限。
一、为人员上下、货物装卸、停车或汽车存放、修理而在道路外设置的设施或场所的汽车出入口起三米以内的部分；
二、道路施工期间，该施工区域侧端起五米以内的部分；
三、消防器械存放处、消防储水槽侧端或其道路出入口起五米以内的部分；
四、消火栓、指定消防水源标识设置位置或消防储水槽吸水口、吸水管投放孔起五米以内的部分；
五、火灾报警器起一米以内的部分。
2. 车辆依据第四十七条第二项或第三项规定停车时，若车辆右侧道路上无三点五米以上的空间（道路标识等指定距离时，指该距离），不得停车。但装卸货物时驾驶人未离开车辆，或驾驶人虽离开但可立即驾驶，或因救助伤病员不得已的情形，不在此限。
3. 在公安委员会认定交通不繁忙并指定的区域，不适用前项本文的规定。

（罚则 第一及二项：第一百十九条之二之四第一项第一号、同条第三项、第一百十九条之三第一项第一号、同条第三项）

（老年驾驶人等标识汽车的临时停车或停车特例）
第四十五条之二 符合下列各项规定的人员（以下本项及次项中称为“老年驾驶人等”）驾驶的普通汽车（仅限该老年驾驶人等依据内阁府令规定，向其住所地管辖的公安委员会申报的），且该老年驾驶人等将依据同项规定领取的老年驾驶人等标识在临时停车或停车期间张贴于车辆前方易见位置的（以下称为“老年驾驶人等标识汽车”），若道路标识等允许在依据第四十四条第一项规定禁止临时停车及停车的道路部分或前条第一项规定禁止停车的道路部分的全部或部分临时停车或停车，不受这些规定限制，可临时停车或停车。
一、持有第七十一条之五第三项规定的普通汽车对应驾照（以下本条中简称为“普通汽车对应驾照”）且年满七十周岁的；
二、符合第七十一条之六第二项或第三项规定的；
三、除前两项所列人员外，持有普通汽车对应驾照，因怀孕或其他事由导致身体机能受限，其驾驶的普通汽车在临时停车或停车地点方面需要特别照顾，且符合政令规定的。
2. 公安委员会应根据老年驾驶人等的申请，向其发放表明该人员为驾驶前项申报的普通汽车的老年驾驶人等的老年驾驶人等标识。
3. 领取老年驾驶人等标识的人员，若该标识遗失、灭失、污损或损坏，可向其住所地管辖的公安委员会申请补发。
4. 领取老年驾驶人等标识的人员，若普通汽车对应驾照被吊销或失效、第一项第三号规定的事由消失，或发生其他内阁府令规定的事由，必须立即将该老年驾驶人等标识返还给其住所地管辖的公安委员会。
5. 除前三项规定外，与老年驾驶人等标识相关的必要事项，由内阁府令规定。

（罚则 第四项：第一百二十一条第一项第十号）

（禁止临时停车或停车地点的特例）
第四十六条 除前条第一项规定外，车辆若道路标识等允许在依据第四十四条第一项或第四十五条第一项规定禁止临时停车及停车的道路部分或禁止停车的道路部分的部分临时停车或停车，不受这些规定限制，可临时停车或停车。

（临时停车或停车的方式）
第四十七条 车辆因人员上下或货物装卸临时停车时，必须尽量沿道路左侧端停放，且不得妨碍其他交通。
2. 车辆停车时，必须沿道路左侧端停放，且不得妨碍其他交通。
3. 车辆在车道左侧端相邻设有路肩（不含以道路标识划分禁止在该路肩临时停车及停车的部分及政令规定的部分）的地点临时停车或停车时，不受前两项规定限制，依据政令规定，可进入该路肩停放，且不得妨碍其他交通。

（罚则 第一项：第一百十九条之三第一项第四号 第二及三项：第一百十九条之二之四第一项第二号、第一百十九条之三第一项第四号）

（临时停车或停车方式的特例）
第四十八条 车辆若道路标识等指定了临时停车或停车方式，不受前条规定限制，必须按照该方式临时停车或停车。

（罚则 第一百十九条之二之四第一项第一号、同条第三项、第一百十九条之三第一项第一号、同条第三项）

（限时停车区段）
第四十九条 公安委员会应对经道路标识等指定为同一车辆可在限定时间内连续停车的道路区段（以下称为“限时停车区段”），为确保该限时停车区段内停车的规范有序，设置并管理停车计时器（仅限具备内阁府令规定功能的，以下同）或停车票发放设备（指具备内阁府令规定功能，用于发放停车票（指内阁府令规定样式的标识，标明领取时间及其他内阁府令规定事项）的设备，以下称为“停车票发放设备”）。
2. 除前项规定外，公安委员会必须向试图在限时停车区段停车的车辆驾驶人提供信息、整理该区段内停放的车辆，并采取其他必要措施确保该限时停车区段内停车的规范有序。
3. 公安委员会可将第一项规定的停车计时器及停车票发放设备的管理相关事务，以及前项规定的措施相关事务的全部或部分，委托给内阁府令规定的人员。

（老年驾驶人等专用限时停车区段）
第四十九条之二 公安委员会可将限时停车区段指定为仅限同一老年驾驶人等标识汽车在限定时间内连续停车的道路区段。此时，公安委员会应在第四十九条第一项规定的道路标识等上标明该事项。

（限时停车区段的停车方式等）
第四十九条之三 限时停车区段内的车辆停车（不含第四十四条第二项各号所列情形下的该客运汽车、无轨电车或该用于旅客运输的汽车的停车，次条同），不受第四十四条至第四十八条规定限制，依照本条至第四十九条之五规定执行。
2. 车辆（在前条规定指定的道路区段（次条中称为“老年驾驶人等专用限时停车区段”）内，仅限老年驾驶人等标识汽车，以下本条、第四十九条之六及第一百十九条之三第一项第二号同）在限时停车区段内，自该停车被第四十九条第一项规定的停车计时器感应时或领取该条第一项规定的停车票发放设备发放的停车票时起，不得超过道路标识等标明的时间连续停车。
3. 车辆在限时停车区段内，必须在道路标识等指定的道路部分，按照指定的方式停车。
4. 车辆驾驶人在限时停车区段内停车后，必须依据政令规定，立即启动第四十九条第一项规定的停车计时器，或立即领取该条第一项规定的停车票发放设备发放的停车票，并在该车辆停车期间（仅限自领取停车票时起至道路标识等标明的时间届满时止），将其张贴于车辆前方易见位置。

（禁止非老年驾驶人等标识汽车在老年驾驶人等专用限时停车区段停车）
第四十九条之四 老年驾驶人等专用限时停车区段内，除老年驾驶人等标识汽车外，其他车辆不得停车。

（罚则 第一百十九条之二之四第一项第一号、同条第三项、第一百十九条之三第一项第一号、同条第三项）

（限时停车区段停车的特例）
第四十九条之五 警察署长依据公安委员会规定，对限时停车区段内的车辆停车指定了可停车地点、停车方式及可开始停车时间、应结束停车时间并予以许可的，若该许可涉及的车辆按照指定的地点、方式，在指定的可开始停车时间至应结束停车时间内开始停车，则该车辆及其驾驶人不适用前两条（不含第四十九条之三第一项）规定。此时，该车辆不得超过指定的应结束停车时间继续停车。

（罚则 后段：第一百十九条之三第一项第一号、同条第三项）

（限时停车区段临时停车的特例）
第四十九条之六 车辆在第四十九条之三第三项规定的道路标识等指定为车辆可停车的道路部分的限时停车区段内，属于第四十四条第一项各号所列的道路部分时，不受该项规定限制，可临时停车。

（限时停车区段内路侧停车场的特例）
第四十九条之七 限时停车区段内设有依据停车场法（昭和三十二年法律第一百零六号）第五条第一项规定的同法第二条第一号规定的路侧停车场（以下本条及第一百十条之二同）时，该路侧停车场涉及的道路部分不适用第四十九条规定。
2. 限时停车区段内设置的路侧停车场涉及的道路部分中，若路侧停车场管理者依据停车场法第六条第一项规定设置了停车计时器或停车票发放设备，则将该停车计时器或停车票发放设备视为第四十九条第一项规定的停车计时器或停车票发放设备，适用第四十九条之三规定。
3. 限时停车区段内设置的路侧停车场涉及的道路部分中，未设置停车计时器或停车票发放设备的，不适用第四十九条之三至第四十九条之五规定。

（禁止进入交叉路口等）
第五十条 试图进入正在进行交通疏导的交叉路口的车辆等，若根据前方行驶车辆等的状况，进入交叉路口（交叉路口内设有道路标识等规定的停止线时，指越过该停止线的部分，以下本项同）后可能在交叉路口内停车，从而妨碍交叉道路上车辆等通行的，不得进入该交叉路口。
2. 车辆等若根据前方行驶车辆等的状况，进入人行横道、自行车横穿带、铁路道口或道路标识划分的部分后可能在该部分停车的，不得进入这些部分。

（罚则 第一百二十条第一项第五号、同条第三项）
